Picsight

Picsight is a photo exploration and image sharing platform created for creators, visual designers, and photographers looking for inspiration, featuring high-fidelity photo layouts and user portfolios.

Technologies Used

React Node.js Express MongoDB Tailwind CSS Bootstrap

Core Features

JWT Authentications

Robust JSON Web Token logins with secure local storage tracking.

Photo Publishing

Easily upload high-quality photography with tag filters and categories.

Comments & Reaction Likes

Engage on photos with clean commenting workspace and heart react triggers.

Visual Search engine

Quickly find creator channels or tags through live discovery queries.

Follow Circle

Follow profiles to customize a personalized home dashboard feed.
