Auctions & C2C

Bidsell

Bidsell is a flexible C2C e-commerce platform that allows users to sell and buy items through either real-time auctions or fixed-price listings. It includes a built-in negotiation chat workspace.

Visit Live Demo

Technologies Used

Firebase DB/Auth Vue 3 Tailwind CSS

Core Features

Firebase Backend Service

Real-time data synchronization for chat messaging and bidding activity.

Auctions & Bidding Engine

Run live auctions with auto-updating highest bid and bidding history logs.

Negotiation Live Chat

Realtime buyer-seller workspace to chat, make custom offers, and negotiate pricing.

Social Authentications

Easy login integrations with Google and Facebook credentials.

Dynamic UI Search & Filtering

Instant listings search with filters for categories, status, and prices.

Three-Language Localization

Fully localizable in Kurdish, English, and Arabic (RTL support).
