Social Platform

Avacial

Avacial is a features-rich social media network designed with a clean UX, bringing together photo-sharing, video-sharing, text micro-blogging, and interactive polling into a seamless flow.

Technologies Used

Vue 3 TypeScript Tailwind CSS Bootstrap Laravel Sanctum MySQL

Core Features

Secure Sanctum API Auth

Robust backend session and API token authentication.

Advanced Publishing Options

Share single videos, photos, image slideshows, markdown text, or interactive polls.

Direct Messaging

Private, secure real-time messaging workspace between users.

Full Profile Controls

Enable private accounts, block list, report bad content, or custom settings.

Dynamic Dark Mode

Easily toggle between beautiful light and dark modes.

Multi-Language Support

Fully localizable interface: English, Kurdish, Arabic.

Admin Moderator Panel

Monitor website content, edit details, view reports and resolve moderation tickets.

Browse screenshots. Click to open full details and zoom.
