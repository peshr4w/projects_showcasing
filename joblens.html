Joblens

Joblens is a specialized hiring platform designed to revolutionize the way job seekers and employers connect, simplifying job discovery and applicant tracking.

Technologies Used

Vue 3 Node.js Express MongoDB Tailwind CSS

Core Features

JWT Authentication Logins

Secure user credentials encryption with JSON Web Token sessions.

Employer Posting Tools

Intuitive forms to create, edit, publish, and manage job listings.

Advanced Candidate Search

Live search engine for filtering jobs by keywords, experience, and tag filters.

Profile Resumes

Construct detailed candidate CV profiles showing experience and skills.

Realtime Alerts Inbox

Receive instant updates regarding applicant status changes or newly posted listings.
